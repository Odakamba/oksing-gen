# OKSING GEN Rule Test Cases

These are the core regression examples for the first public version.

## Prokem

```text
aku telat sekolah, aku beli roti
-> okak tokel skokul, okak bokel rokot

kamu kemana jarang pulang
-> okkam kokem jokar pokul
```

## Prinem

```text

-> kinaminu minakinan rinotini

-> diniina sinukina binakpinao

gimana sih, kan itu salah lu
```
