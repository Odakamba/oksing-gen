# OKSING GEN

Prokem, Prinem, Pregem & Presm Translator by Oda F.T.

OKSING GEN is a static web app for translating Indonesian text into four OKSING language modes and back again:

- Prokem
- Prinem
- Presm
- Pregem

The app also includes:

- reverse translation with fuzzy suggestions
- a zine/embed page for OKSING history
- a quiz mode for practice
- a black and gold visual theme

## Deploy

This project is static. Upload the `oksing-gen` folder to GitHub and deploy it on Vercel as a static project.

No build step is required.

See [DEPLOY.md](./DEPLOY.md) for the exact Vercel settings.

## Local Preview

Open `index.html` directly, or serve the folder with any static server.
